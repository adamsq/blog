建立hexo站点

安装node.js

在使用hexo前必须先架设node.js环境。在 http://nodejs.org/ 上可以下载node.js的最新源码，然后使用make install 进行编译，即可在系统里生成node.js的环境。

$ ./configure
$ make install

安装hexo

使用npm命令直接安装hexo及它的命令行工具

建立站点

使用 http://hexo.io 上的例子可以快速创建一个hexo站点

$ hexo init blog
$ cd blog
$ npm install
$ hexo server

之后使用 http://localhost:4000 就可以在本地访问站点

修改hexo站点

修改主题

本站用的是 http://hexo.io 上theme页的jacman主题，先从git上下载下来，然后修改hexo目录下_的文件即可设置新主题。

新建文章

用hexo new命令可以创建一篇新的文章，然后用编辑器编辑即可，支持Markdown语法

与github同步

总结

  • hexo依赖于node.js，在安装前必须先架设node.js环境。
  • hexo内置与github服务器同步的命令，前提是安装好git，并在配置文件下设置git的相关配置。
  • 使用hexo server命令可以在本地查看站点，再用hexo d命令与github进行一键同步。
  • 用hexo new命令创建文章，并用Markdown语法直接编辑文件。
